Масштаб задачи: 10 deliverables

Это несколько независимых подсистем. Давайте определим, что делаем сначала.

Фаза 1 — Фундамент
📐 Slide Scene Graph

SlideScene, SceneNode, BackgroundConfig — новая модель данных

🎨 Theme / Token System

Дизайн-токены вместо 5 свойств, semantic color roles

🏗️ Layout Planner (12 шаблонов)

Детерминированные шаблоны + правила плотности

🤖 AI Contract

AI генерирует semantic intent, движок — layout/spacing

Фаза 2 — Редактор
🖼️ Composition Engine

Auto-layout, baseline grid, focal point, safe zones

✏️ Slide Editor UI

Navigator, layer panel, snapping, smart guides

⭐ Visual Quality Evaluator

Scoring: balance, whitespace, readability, hierarchy

Фаза 3 — Полировка
🎬 Animation System

Timeline, transitions, progressive disclosure

🎭 Presenter Preview

Speaker view, timer, notes overlay

Фаза 4 — Миграция и экспорт
📦 Export Pipeline

PDF, PPTX, HTML — через новый движок

🔄 Migration

Конвертация старых SlideGraph → SlideScene

Моя рекомендация:

Начать с Фазы 1 — это фундамент, от которого зависит всё остальное. Фаза 2 даёт полноценный редактор. Фазы 3–4 можно делать итеративно. Каждая фаза получает отдельный spec → plan → реализацию.
