Контекстная панель + Quality Scorer

Панель меняется по типу выбранного элемента. Оценка считается локально, без AI.

Логика переключения панели
// ContextPanel.tsx
function ContextPanel() {
const { selectedElemId, rightMode }
= useEditorStore()
const elem = useSlideStore(
s => findElement(s, selectedElemId)
)

// rightMode из иконок сверху
if (rightMode === 'slide')
return <SlideProperties />
if (rightMode === 'theme')
return <ThemeEditor />
if (rightMode === 'notes')
return <NotesPanel />

// props mode — по типу элемента
if (!elem)
return <SlideProperties />

return <ElementProperties
elem={elem} />
}
Секции по типу элемента
Текст
heading · subheading · body · quote · caption · label
Содержимое (textarea) → Шрифт + размер + вес → Стиль (B/I/U) → Выравнивание → Цвет → Межстрочный
Список
bullet-list
Пункты (добавить/удалить/переупорядочить) → Стиль маркера → Шрифт → Цвет
Изображение
image
URL / загрузка → Alt-текст → Object-fit (cover/contain) → Скругление → Прозрачность
Карточки
metric-card · feature-card · step-card · pricing-card
Поля по типу (label/value/description/badge) → Фон → Скругление
Общее для всех типов (всегда снизу)
Позиция X/Y · Размер W/H · Прозрачность · Имя слоя · 🔒 Lock · 👁 Visible
Quality Scorer — 4 метрики, считается локально
Плотность
Суммарная площадь элементов / площадь слайда. Штраф если > 65% или < 20%
72 / 100
Иерархия
Есть ли heading? Разница размеров heading vs body ≥ 1.4×? Один heading или больше?
90 / 100
Баланс
Центр масс элементов vs центр слайда. Отклонение > 15% — штраф
60 / 100
Читаемость
heading ≥ 28px, body ≥ 14px. Контраст цвета текста vs фона (WCAG AA)
85 / 100
Итоговый балл
score = (
  density * 0.25 +
  hierarchy * 0.35 +
  balance * 0.20 +
  readability * 0.20
) → 0..100
Когда считается
• При каждом изменении слайда
• Debounce 300ms
• Только текущий слайд
• Без сетевых запросов
• < 1ms на вычисление
