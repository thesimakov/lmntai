Поток данных
SlideGraph (Zustand store)
Единственный источник истины. Все изменения идут через actions: updateElement(), moveElement(), addSlide()
↓ читает
SlideCanvas (960×540)
Рендерит элементы как React-компоненты. Масштабируется через transform: scale(ratio) где ratio = containerWidth / 960
↑ пишет
InteractionLayer (поверх холста)
Абсолютно позиционирован поверх SlideCanvas в тех же координатах. Обрабатывает drag, resize, click — пишет в store через actions
↕ синхронизирует
ContextPanel (реагирует на selection)
Читает selectedElementId из UI-store, показывает нужные контролы, пишет изменения через те же actions
CSS-scale математика (ключевой момент)
Холст всегда 960×540
const CANVAS_W = 960
const CANVAS_H = 540

const scale = containerW / 960

<div style={{
  width: 960,
  height: 540,
  transform: `scale(${scale})`,
  transformOrigin: 'top left'
}}>
Координаты мыши → холст
function toCanvas(e) {
  const rect =
   canvasRef.getBoundingClientRect()
  return {
   x: (e.clientX - rect.left) / scale,
   y: (e.clientY - rect.top) / scale
  }
}
Snap grid (8px)
const SNAP = 8

function snap(v: number) {
  return Math.round(v / SNAP) * SNAP
}
