GrapesJS — какие улучшения включаем?

Выбери одно или несколько (multi-select). Zero Block: настройки tooltip/form/gallery и брейкпоинты 480/320 — уже в скопе.

А

Фон секции из тулбара

Кнопка 🎨 в тулбаре выбранной секции → попап с выбором цвета / градиента / фонового изображения. Без открытия правого Style Manager.

Секция с выбранным фоном
Б

Inline double-click редактирование текста

Двойной клик по тексту внутри секции → contenteditable прямо на холсте. GrapesJS уже поддерживает RTE, нужно настроить и локализовать панель форматирования.

Выделенный текст здесь
В

Ctrl+Shift+V — вставить HTML секцию

Буфер обмена содержит HTML → вставляется как новая секция в конец холста. Полезно при переносе секций между проектами без библиотеки блоков.

Г

Счётчик и навигация по секциям

Внизу холста: «Секция 3 / 7». Стрелки ← → прокручивают к предыдущей / следующей секции. Полезно на длинных страницах.

Д

Delete / Backspace = удалить выбранную секцию

Когда выбрана секция (не текстовый элемент) — клавиша Delete сразу удаляет её без подтверждения. Аналог Figma/Tilda.

Можно выбрать несколько. Если нужно что-то другое — напиши в терминале.
