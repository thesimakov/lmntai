Зеркало saveweb2zip лежит здесь (как у исходника): css/, js/, fonts/, images/, index.html.

Файл grapes-bootstrap.json генерируется скриптом (инлайн CSS + тело страницы с путями под /box-templates/wpriver-saveweb/):

  npm run build:wpriver-bootstrap

После замены index.html или CSS в этой папке перезапустите команду выше.
